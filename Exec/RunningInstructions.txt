> Usage:
> Parameters:
> -fa         [./Reference reads directory in fasta format]
                 Required parameter if no index file is provided.

> -vpt_load   [./Index file generated by the CBN software]
                 Required parameter if no references fasta file is provided.

> -r          [./Short reads directory in fasta or fastq format]
                  Required parameter.

> -o          [Directory to save classification results]
                  Optional parameter.
                  If a directory is not provided results will be stored in the same directory as the short reads file.

> -kmer       [Integer value indicating the kmer size to be used to build the tree]
                  Optional parameter. (Default value: 64)

> -rep        [String value indicating the representation method to be used to build the tree]
                  Optional parameter. (Default value: Tetrahedron)
                  Options:
                          Atomic_Numbers
                          Complex_Numbers
                          Dna_Walk
                          EIIP_numbers
                          Integer_numbers
                          Pair_numbers
                          Real_numbers
                          Tetrahedron
                          Voss_indicators
                         Z_curve

> -tra        [String value indicating the transformation method to be used to build the tree]
                  Optional parameter. (Default value: DWT)
                  Options:
                          DFT
                          DWT
                          PAA

> -clvl       [Integer value indicating the data compression levelt o be used to build the tree]
                  Optional parameter. (Default value: 4)
> -KNN        [Integer value indicating the number of K nearest neighbours to report]
                  Optional parameter. (Default value: 5)
> -ra         [Double value indicating the length to be used for the range search]
                  Optional parameter. (Default value: 0.5)
> -sen        [String value indicating the search strategy to be used for the KNN search.]
                  If sen is set to yes a more sensitive but slowr KNN search is performed wheres if
                  sen is set to no a more faster but not as sensitive KNN search is performed 
                  Optional parameter. (Default value: No)
                          y or Y or Yes
                          n or N or No

> -evs        [String value indicating the speed of  the evalaution speed of the knn results]
                  Optional parameter. (Default value: M)
                  Options:
                          F or Fast   [Less accurate]
                          M or Medium [Accurate]
                          S or Slow   [The most rigorous] 

> -proc       [String value indicating if a preprocess should be performed]
                  Optional parameter. (Default value: No)
                  Options:
                          y or Y or Yes
                          n or N or No

> -s          [String value indicating if the tree needs to be stored on hard drivet]
                  Optional parameter. (Default value: No)
                  Options:
                          y or Y or Yes
                          n or N or No

> -vpt_save   [./directory to save index file]
                  Optional parameter.
                  If a directory not provided results will be stored in the same directory as the Reference reads file.
                  If an index structure is provided for the search, the tree will not be saved.

> -accu       [String value indicating if a representations accumulated sum should be performed]
                  Optional parameter. (Default value: No)
                  Options:
                          y or Y or Yes
                          n or N or No

> -znorm      [String value indicating if a representations Z-normalisation should be performed]
                  Optional parameter. (Default value: No)
                  Options:
                          y or Y or Yes
                          n or N or No

> -ptd        [String value indicating if tree details should be printed]
                  Optional parameter. (Default value: No)
                  Options:
                          y or Y or Yes
                          n or N or No

> -repall     [String value indicating if details for all reads (including unmatched reads) should be saved or only matched reads details should be saved 
                  Optional parameter. (Default value: Yes)
                  Options:
                          y or Y or Yes
                          n or N or No

> -h or -help              Print the instructions manual.
Examples:
ClassificationByNumbers -fa ./References.fasta -r ./ShortReads.fastq -o ./OutputDirectory
ClassificationByNumbers -vpt_load ./References.vptstruct -r ./ShortReads.fastq -o ./OutputDirectory
ClassificationByNumbers -fa ./References.fasta -r ./ShortReads.fastq -o ./OutputDirectory -kmer 32 -rep Voss_indicators -tra PAA -clvl 8 -KNN 10 -speed F -proc Y -s Y -vpt_save ./Save_IndexDirectory/ -repall Yes 
ClassificationByNumbers -fa ./References.fasta -r ./ShortReads.fastq -o ./OutputDirectory -kmer 32 -rep Voss_indicators -tra PAA -clvl 8 -ra 0.25 -speed F -proc Y -s Y -vpt_save ./Save_IndexDirectory/ -repall No
ClassificationByNumbers -vpt_load ./References.vptstruct -r ./ShortReads.fastq -o ./OutputDirectory -kmer 32 -rep Voss_indicators -tra PAA -clvl 8 -KNN 10 -speed F -proc Y -repall Yes
ClassificationByNumbers -vpt_load ./References.vptstruct -r ./ShortReads.fastq -o ./OutputDirectory -kmer 32 -rep Voss_indicators -tra PAA -clvl 8 -ra 0.25 -speed F -proc Y -repall No
